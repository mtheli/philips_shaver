# Philips Shaver Integration for Home Assistant

Custom Home Assistant integration for Philips Shaver devices.

## Installation via HACS

1. Add this repository to HACS under `Custom Repositories`.
2. Choose the release (e.g., `0.1.0`) or the `main` branch.
3. Install and configure via Home Assistant UI.

## Requirements

- Home Assistant 2025+
- BLE capable hardware (USB or built-in adapter)
